# Usage

<code>
$.kbd({
 white:false, //keyboard theme (def.: false(=black))
 qwertz:false, //layout (def.: false(=qwerty))
 hidden:false, //just init, don't show
 input:function(char){[...]}, //input callback
 coffee:false //coffee splash? (white theme)
});
</code>
